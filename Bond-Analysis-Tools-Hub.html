How to Choose

1. Smart Evaluator: Best for making a final Buy/Hold/Sell decision. Uses an intelligent weighted scoring model to give a clear recommendation.

2. Analysis Flowchart: Best for quick decisions based on critical gates. Stops analysis as soon as a dealbreaker is found.

3. Valuation Screener: best for detailed calculated valuation. Great for domestic bonds with clear YTM benchmarks.

4. Bond Screener: Best for analyzing international bonds. Accounts for currency depreciation and home inflation.

5. Complete Evaluation: Institutional-grade analysis with 7 phases covering real returns, interest rate risk, and credit risk.

6. Screener Pro: Best for relative scoring. Ranks multiple bonds comparatively using a points-based system.

Methodology

All tools utilize real-time server-side processing to protect proprietary financial logic.

Calculations account for inflation, credit spreads, and duration-matched required yields.
