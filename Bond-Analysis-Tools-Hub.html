How to Use These Tools

1. Start with Analysis Flowchart if you want a quick decision based on critical gates. This tool stops analysis as soon as a dealbreaker is found.

2. Use Valuation Screener if you need detailed valuation analysis with fair price calculation. Great for domestic bonds with clear YTM benchmarks.

3. Choose Bond Screener if you're analyzing international bonds and need to account for currency depreciation and home inflation.

4. Run Complete Evaluation for institutional-grade analysis with 7 phases covering real returns, interest rate risk, and credit risk.

5. Try Screener Pro if you want a scoring system that ranks bonds comparatively. Best for portfolio analysis of multiple bonds.
